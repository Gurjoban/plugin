== Description ==
Footer Text Widget plugin use to put multiple text values in single widget area.

Installation steps

	1. Install the  plugin
        2. Then activate it and read the description
	3. then go to the apperance-widget
	4. Add the widget to the side bar you have
	5. You can add the text that you wanted to show on the footer
	6. It will be appeared on all the pages wherever you have the sidebar
